## 開発動機

AnDropは、AndroidデバイスからmacOSデバイスにwifi経由でファイルを転送するためのアプリです。
最もシンプルで最高のユーザー体験を提供することを目的としています。
つまり、セットアップが不要で、デバイスの検出・転送速度が早いということです。

AnDropは、Android OSのメニューに統合されているため、最小限のUIで動作し、使用時のストレスを軽減します。
